# アウトゲーム仕様

## 画面遷移フロー（仮）
1. タイトル画面
2. ロビー作成 / 参加（ルームID入力方式）
3. オープニングムービー
4. ゲーム開始
5. ゲームクリア / ゲームオーバー
6. リザルト画面

## 決定済み仕様
- **マッチングシステム**: フレンド招待のみ（ルームID入力）。ランダムマッチングなし。
- **設定画面（Option）**:
    - 音量調節（BGM, SE）
    - 言語設定
    - 解像度：固定
- **セーブデータ**: なし（ワンショットプレイ）。中断セーブも基本なし。
- **チュートリアル**: なし。導入は短いムービーのみで世界観を伝える。
